## Learn to use Kubejs to work with Mekanism (no addons) 
**(check next section for addon custom code)**


Lets show you how to use *kubejs* to add 
your own custom recipes with Mekanism this section will include
the `Energized Smelter`, `enriching_factory`


## Creating a Recipe for the `Energized Smelter`

For creating stuff for the `Energized Smelter` You want 
to figure out what you want as your input and your output.
For this one we are going 
for `apples` as our input and a `golden_apple` as
the output

*The Smelting factory types all run off of whats already in minecraft
If it can smelt in a furnace it can smelt in the factories. So nothing to crazy. Just use a normal smelting recipe in your code*

![Smelting facotries](/img/mek/smelting.png)


<div style={{height: '100px'}}></div>

```jsx title="Energized Smelter.js"
ServerEvents.recipes(event => {
    event.custom({
        "type": "minecraft:smelting",
        "ingredient": {
            "item": "minecraft:apple"
        },
        "result": "minecraft:golden_apple",
        "experience": 0.35,
    })
})


```

## Creating a Recipe for the `Rotary Condensentrator`

For creating stuff for `Rotary Condensentrator` You want 
to figure out what Gas you want as your input, and what fluid you want as your output. For this one i have 
the gas uranium_hexafluoride as my input and minecraft:water as my output. If you switch the two around as input and output 
the game will automatically change the recipe to work as Decondensentrating instead!!

![Rotary Condensentrator](/img/mek/Rotary_condensentrator.png)


<div style={{height: '100px'}}></div>

```jsx title="Rotary Condensentrator.js"
ServerEvents.recipes(event => {
    event.custom({
        type: "mekanism:rotary",

        gasInput: {
            amount: 1,
            gas: "mekanism:uranium_hexafluoride"
        },

        fluidOutput: {
            amount: 1,
            fluid: "minecraft:water"
        }
    })
})
```

## Creating a Recipe for the `Pressurized Reaction Chamber`

For creating stuff for `Pressurized Reaction Chamber` You want a fluid as your first main input mine here is `lava`
and then you want a gas of some sort, mine here is `Polonium`. After that you want a item input and here it is a `Apple` and the output for me is `golden_apple`

![Pigment_mixing](/img/mek/Pressurized+Reaction_Chamber.png)


<div style={{height: '100px'}}></div>

```jsx title="Pressurized+Reaction_Chamber.js"

    event.custom({
        type: "mekanism:reaction",
        duration: 100,
        fluidInput: {
            amount: 1000,
            tag: "minecraft:lava"
        },
        gasInput: {
            amount: 1000,
            gas: "mekanism:polonium"
        },
        itemInput: {
            ingredient: {
                item: "minecraft:apple"
            }
        },
        gasOutput: {
            amount: 1000,
            gas: "mekanism:spent_nuclear_waste"
        },
        itemOutput: {
            item: "minecraft:golden_apple"
        }
    })
})
```
