Learn Mekanism forge 1.20.X

Learn to use Kubejs to work with Mekanism (no addons)​

(check next section for addon custom code)

Lets show you how to use kubejs to add your own custom recipes with Mekanism this section will include the Energized Smelter, enriching_factory

Creating a Recipe for the Energized Smelter​

For creating stuff for the Energized Smelter You want to figure out what you want as your input and your output. For this one we are going for apples as our input and a golden_apple as the output

The Smelting factory types all run off of whats already in minecraft If it can smelt in a furnace it can smelt in the factories. So nothing to crazy. Just use a normal smelting recipe in your code

Energized Smelter.js
ServerEvents.recipes(event => {
    event.custom({
        "type": "minecraft:smelting",
        "ingredient": {
            "item": "minecraft:apple"
        },
        "result": "minecraft:golden_apple",
        "experience": 0.35,
    })
})

Creating a Recipe for the Rotary Condensentrator​

For creating stuff for Rotary Condensentrator You want to figure out what Gas you want as your input, and what fluid you want as your output. For this one i have the gas uranium_hexafluoride as my input and minecraft:water as my output. If you switch the two around as input and output the game will automatically change the recipe to work as Decondensentrating instead!!

Rotary Condensentrator.js
ServerEvents.recipes(event => {
    event.custom({
        type: "mekanism:rotary",

        gasInput: {
            amount: 1,
            gas: "mekanism:uranium_hexafluoride"
        },

        fluidOutput: {
            amount: 1,
            fluid: "minecraft:water"
        }
    })
})

Creating a Recipe for the Electrolytic seperator​

For creating stuff for Electrolytic seperator You want to figure out what Fluid you want as your input, and what fluid or gas you want as your output. For this one i have the fluid minecraft:water as my input and gas:chlorine and gas Oxygen as my output.

Electrolytic_seperator.js
ServerEvents.recipes(event => {
    event.custom({

    "amount": 2,

  },
  "leftGasOutput": {
    "amount": 2,
    "gas": "mekanism:chlorine"
  },
  "rightGasOutput": {
    "amount": 1,
    "gas": "mekanism:oxygen"
  }
    })
})

Creating a Recipe for the Chemical dissolution chamber​

For creating stuff for Chemical dissolution chamber You want to figure out what gas you want as your main input. Mine is sulfuric asid, and the item input is iron ore. And the output here will be a custom output of slurry this one is completely custom from Mekanism

Chemical_dissolution_chamber.js
ServerEvents.recipes(event => {
    event.custom({
  "type": "mekanism:dissolution",
  "gasInput": {
    "amount": 1,
    "gas": "mekanism:sulfuric_acid"
  },
  "itemInput": {
    "ingredient": {
      "item": "minecraft:iron_ore"
    }
  },
  "output": {
    "amount": 1000,
    "chemicalType": "slurry",
    "slurry": "mekanism:dirty_copper"
  }
})
})

Creating a Recipe for the Pigment Mixing​

For creating stuff for Pigment Mixing You want to figure out what 2 pigments you want as your main inputs. Mine is black and yellow. And the output here will be a green. Yeah i know it makes no sense but hey, thats the fun of custom code

Pigment_mixing.js
ServerEvents.recipes(event => {
    event.custom({
        type: "mekanism:pigment_mixing",
        leftInput: {
            amount: 1,
            pigment: "mekanism:black"
        },
        rightInput: {
            amount: 4,
            pigment: "mekanism:yellow"
        },
        output: {
            amount: 5,
            pigment: "mekanism:green"
        }
    })
})

Creating a Recipe for the Chemical Washer​

For creating stuff for Chemical Washer You want to figure out what chemical you want as your main inputs. And then use water as your other main input. And the output here will be a different chemical Its up to you what all you put in here. Any chemical to be washed, and any fluid as the cleaning supply. and any chemical as the output.

Chemical_Washer.js
ServerEvents.recipes(event => {
    event.custom({
        type: "mekanism:washing",
        fluidInput: {
            amount: 5,
            tag: "minecraft:water"
        },
        slurryInput: {
            amount: 1,
            slurry: "mekanism:dirty_iron"
        },
        output: {
            amount: 1,
            slurry: "mekanism:clean_copper"
        }
    })
})

Creating a Recipe for the Pressurized Reaction Chamber​

For creating stuff for Pressurized Reaction Chamber You want a fluid as your first main input mine here is lava and then you want a gas of some sort, mine here is Polonium. After that you want a item input and here it is a Apple and the output for me is golden_apple

Pressurized+Reaction_Chamber.js
ServerEvents.recipes(event => {
    event.custom({
        type: "mekanism:reaction",
        duration: 100,
        fluidInput: {
            amount: 1000,
            tag: "minecraft:lava"
        },
        gasInput: {
            amount: 1000,
            gas: "mekanism:polonium"
        },
        itemInput: {
            ingredient: {
                item: "minecraft:apple"
            }
        },
        gasOutput: {
            amount: 1000,
            gas: "mekanism:spent_nuclear_waste"
        },
        itemOutput: {
            item: "minecraft:golden_apple"
        }
    })
})
