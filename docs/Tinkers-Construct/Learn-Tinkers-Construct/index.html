Learn-Tinkers-Construct

Tinkers Construct KubeJS Overview​

This page contains working KubeJS examples for modifying Tinkers' Construct recipes in Minecraft 1.20.1. All examples are tested and ready to use as templates for custom modpacks.

Requirements​

  • Minecraft 1.20.1
  • Forge 47.4.10
  • Tinkers' Construct 3.11.2.166
  • KubeJS installed and working

Notes​

  • Some recipe types may change between Tinkers versions
  • Not all recipes work in all modpack setups
  • Always test recipes in-game after adding them

Common Fields (Melting / General Recipe Reference)​

FieldDescription
ingredientThe item being used in the recipe.
fluidThe fluid produced from the recipe.
amountHow much fluid is created.
temperatureRequired heat level to process the item.
timeHow long the melting process takes.

Creating a Recipe for the Melter / Smeltery​

For creating stuff for Melter / Smeltery is just taking an item and melting it down into a fluid. Melting Recipes are used to determine what items can be melted, what fluid they produce, how much fluid is created, the temperature required to melt the item, and how long the melting process takes. This is commonly used for creating custom molten materials, recycling items into fluids, or integrating custom items into the Tinkers' Construct smelting system.

Melter_Smeltery.js
ServerEvents.recipes(event => {
    event.custom({
        type: "tconstruct:melting",
        ingredient: {
            item: "minecraft:golden_apple"
        },
        result: {
            fluid: "minecraft:water",
            amount: 1000
        },
        temperature: 300,
        time: 50
    })
})

Creating a Recipe for the Casting Table​

For creating stuff for Casting Table is just pouring a fluid onto a cast or item to create a new item. Casting Table Recipes are commonly used for creating tool parts, ingots, nuggets, gems, and other small items from molten fluids. These recipes allow you to define the required fluid, the cast being used, whether the cast is consumed during the process, and how long the item takes to cool before it can be collected.

Casting_Table.js
ServerEvents.recipes(event => {
    event.custom({
        type: "tconstruct:casting_table",
        cast: {
            item: "minecraft:stick"
        },
        cast_consumed: false,
        fluid: {
            fluid: "minecraft:water",
            amount: 1000
        },
        result: "minecraft:golden_apple",
        cooling_time: 60
    })
})

Creating a Recipe for the Casting Basin​

For creating stuff for Casting Basin is just pouring a fluid into a basin to create a larger item or block. Casting Basin Recipes are typically used for creating storage blocks, metal blocks, glass blocks, and other large casted objects. These recipes allow you to define the fluid being used, the resulting item, and the cooling time required before the finished item can be removed from the basin.

Casting_Basin.js
ServerEvents.recipes(event => {
    event.custom({
        type: "tconstruct:casting_basin",
        fluid: {
            fluid: "minecraft:water",
            amount: 1000
        },
        result: {
            item: "minecraft:golden_apple"
        },
        cooling_time: 100
    })
})

Creating a Recipe for the Alloyer​

For creating stuff for Alloyer is just combining two or more fluids together to create an entirely new fluid. Alloy Recipes are used to create molten alloys by mixing specific fluids in defined amounts. This system is commonly used for creating advanced materials and custom metals that cannot be obtained through normal melting recipes. The resulting fluid can then be used in casting recipes or other Tinkers' Construct processing systems.

Alloyer.js
ServerEvents.recipes(event => {
    event.custom({
        type: "tconstruct:alloy",
        inputs: [
            {
                fluid: "minecraft:water",
                amount: 500
            },
            {
                fluid: "minecraft:lava",
                amount: 500
            }
        ],
        result: {
            fluid: "minecraft:obsidian",
            amount: 1000
        }
    })
})

Creating a Recipe for the PART BUILDER/Material Recipe​

For creating stuff for Material Recipe is just taking an item and assigning it a material value that can be used by Tinkers' Construct. Material Recipes are used to tell Tinkers' Construct what items can act as a material and how much of that material they provide when used in recipes. This is useful when adding custom materials or allowing existing items to be used as material sources.

Material_Recipe.js
ServerEvents.recipes(event => {
    event.custom({
        type: "tconstruct:material",
        ingredient: {
            item: "minecraft:golden_apple"
        },
        material: "tconstruct:wood",
        value: 1,
        needed: 1
    })
})

Creating a Recipe for the Modifier Recipe​

For creating stuff for Modifier Recipe is just taking an item and applying a modifier to a compatible Tinkers' Construct tool. Modifier Recipes are used to add upgrades and abilities to tools, weapons, and armor. In this example, using a Golden Apple will apply the Luck modifier to any tool that can accept modifiers. This allows pack developers to create custom upgrade paths, progression systems, or entirely new ways of improving Tinkers' Construct equipment.

Modifier_Recipe.js
ServerEvents.recipes(event => {
    event.custom({
        type: "tconstruct:modifier",
        tools: {
            tag: "tconstruct:modifiable"
        },
        inputs: [
            {
                item: "minecraft:golden_apple"
            }
        ],
        result: "tconstruct:luck"
    })
})
