# Removing Recipes

Removing unwanted recipes is very common in modpacks. All removal scripts go in the **`kubejs/server_scripts/`** folder.

```js
// File: server_scripts/remove_recipes.js
ServerEvents.recipes(event => {
    event.remove// Your removal code goes here
});
```



This is specifically what you need to **`remove`** recipes in bulk, sings, by tags, and so on. Check the lower examples to see more information
```js
event.remove
```

## Removing outputs of crafting items
**`Example 1, Removing the output of a item being crafted`**

![My Logo](/img/remove/diamond_pickaxe_removed.png)

```js
// File: server_scripts/remove_recipes.js
ServerEvents.recipes(event => {
    // Remove all ways to craft a Diamond Pickaxe
    event.remove({ output: 'minecraft:diamond_pickaxe' })
})
```

## Removing ALL recipes in ANY mod

**`Example 3, This is how you can remove the outputs or recipes of EVERY item, block, or tool that is in a specific mod. This example shows you it working with the Create mod.`**

![My Logo](/img/remove/remove_entire_mod.png)


```js
// File: server_scripts/remove_recipes.js
ServerEvents.recipes(event => {
    // Remove ALL recipes from the Create mod
    event.remove({ mod: "create" })
})
```

## Removing recipes with specific Inputs/Ingredients

**`Example 4, This is how you remove a recipe if it uses a specific item or material as its input or ingredient, This example shows me removing any recipe that uses oak planks as an ingredient, so every block or item in the game that uses oak planks no longer have a crafting recipe`**


![My Logo](/img/remove/remove_based_off_ingredients.png)


```js
// File: server_scripts/remove_recipes.js
ServerEvents.recipes(event => {
    // Remove any recipe that uses Oak Planks as an ingredient
    event.remove({ input: 'minecraft:oak_planks' });
})
```

## Removing ALL types of output for a specific block (the campfire outputs)

**`Example 5, This specific code shows you how to remove the output recipes of a entire block. in this case it shows you that if you do this with the campfire
then you will no longer be allowed to put anything food wise on the camp fire and cook it. It completely removes cooking on the campfire `**


![My Logo](/img/remove/remove_campfire_ALL.png)


```js
// File: server_scripts/remove_recipes.js
ServerEvents.recipes(event => {
    // Remove all campfire cooking recipes
    event.remove({ type: 'minecraft:campfire_cooking' });

})
```

## Removing SPECIF types of output for a specific block (the furnace outputs and blast furnace outputs)

**`Example 5, this example shows that you can remove specific outputs also and not just ALL outputs. this shows that you can disable or remove
smelting a ore or raw ore like raw iron in a furnace but still have it working in ONLY the blast furnace AND this also works for the camp fire`**


![My Logo](/img/remove/blast_furnace_working.png), ![My Logo](/img/remove/furnace_not_working.png)

```js
// File: server_scripts/remove_recipes.js
// Remove all blasting recipes for iron
    ServerEvents.recipes(event => {
    // Remove all campfire cooking recipes
    event.remove({ type: 'minecraft:smelting', output: 'minecraft:iron_ingot' })

})
    
```
