learn basic SHAPED recipes

ServerEvents.recipes(event => {
  event.shaped('minecraft:grass_block', [
    'DD ',
    '   ',
    '   '
  ], {
    D: 'minecraft:dirt'
  })
})

To make a shaped recipe it is pretty much just like the shapless just with a few extra steps.

The biggest thing is event.shaped('minecraft:grass_block', [

This tells minecraft that it is a shaped recipe, and sense there is event.shaped instead event.custom it automatically uses the crafting table.

the code part that looks like

    'DD ',
    '   ',
    '   '

is just telling the game what shape to put the items in the crafting table. Look at spaces inbeteen the '' and think of thoes as just empty spots in the crafting table

This is what we call "Keys" This just gives a name of something to a letter. What ever item you give to this letter is what will be used in the crafting table!!!

More shaped Crafting recipe examples​

There is many different ways you can craft things using chaped crafting, here are three more examples of some fun things to doc

// Vertical Recipe Example
ServerEvents.recipes(event => {
  event.shaped('minecraft:diamond', [
    'I  ',
    'I  ',
    'I  '
  ], {
    I: 'minecraft:iron_ingot'
  })
})

// Corner Recipe Example
ServerEvents.recipes(event => {
  event.shaped('minecraft:gold_ingot', [
    'G  ',
    ' G ',
    '  G'
  ], {
    G: 'minecraft:gold_nugget'
  })
})

// Full Square Recipe Example
ServerEvents.recipes(event => {
  event.shaped('minecraft:emerald', [
    'EEE',
    'EEE',
    'EEE'
  ], {
    E: 'minecraft:emerald_block'
  })
})
