# Learn to format your files

**`There is a LOT of things in kubejs that NEED to be in the right folders. You could have all your code correct
and nothing will work. Things like `**

- recipe scripts
- actual game function changes
- functions on player join
- command creation
- etc


**`Lets start off by learing where you can find your mod files and kubejs files`**

The biggest thing you need to know is how to find your game files. For me im going to show this
in CurseForge but it may differ from launcher to launcher


**`Lets start by actually going into your modpack in CurseForge and right next to the play button you will see 3 little dots`**
![My Logo](/img/open_mod_files.png)

**`Now click on open folder`**


## Entire your mod file folder

**`Now that we are in your modpack directory you need to find a folder called Kubejs, that is where ALL of your work will be done`**

![My Logo](/img/file_layout_beginning.png)


**`Inside the Kubejs folder you will see a few different folders and a ReadMe file, the few files you will more than likely
be worried about will be the server_scripts and startup_scripts. This will be where most of your code will be ran, made, or messed with`**

![My Logo](/img/inside_kubejs_folder.png)

**`Both the Server_Script 
and the startup_scripts folder will 
usually have a file in them by default. 
In different version they may be names something different 
but in this example they are called main.js in the startup_scripts 
and main.js in the server_scripts but are sometimes names example.js`**

![My Logo](/img/inside_startup_script.png)
![My Logo](/img/inside_server_script.png)


## Creating a new .js script

**To start making your own file or script, it is very simple. You can start out by using the example one that is created for you at first launch
but when you are ready to create a NEW file it is very simple.**

**Start out by going into one of the two folders that we spoke about before (startup_scripts/Server_scripts)**


**Start out by right clicking anywhere in the folder and go down to NEW**
![Inside Server Scripts](/img/new_file_doc.jpg)

**then click on new text document**

**Once you get a new file, rename it to something relative to what you are changing and change the .txt to .js**
![Inside Server Scripts](/img/new_text_doc.jpg)
![Inside Server Scripts](/img/renaming_doc.jpg)

**When clicking ENTER it will ask you if you are sure you would like to change the file type from .txt to .js, just click yes**

![Inside Server Scripts](/img/selecting_yes.jpg)
