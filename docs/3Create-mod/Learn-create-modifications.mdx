## MECHANICAL CRAFTERS

**forge 1.20.1**

![mechanical crafter example](/img/c/MECHANICAL_CRAFTER.png)
![mechanical crafter JEI example](/img/c/Mechanical_craftingJEI.png)

*In this example we show you how to make a very unfair recipe, but fun none the less. 
You need to fill up a 4X4 set of mechanical crafters to make 1 single oak plank. Very annoying but this is a great recipe*

```jsx title="mechanical_crafters.js"
ServerEvents.recipes(event => {
    event.custom({
        type: "create:mechanical_crafting",
        pattern: [
            "AAAA",
            "AAAA",
            "AAAA",
            "AAAA"
        ],
        key: {
            A: {
                item: "minecraft:oak_log"
            }
        },
        result: {
            item: "minecraft:oak_planks",
            count: 1
        }
    })
})
```

## Draining

**forge 1.20.1**

![draining example](/img/c/Draining.png)
![draining JEI example](/img/c/drainingJEI.png)


*This recipe will show you how to take a item `IN` to the drain and output two things, 
You absolutly have to have a fluid output. if you want 2 items the second one has to be a item, 
not another fluid. The drain can only hold up to one fluid at a time. 
Just make sure you 100% of the time have some sort of 
`fluid` coming out as a result or you will crash your game and the recipe wont work*



```jsx title="drain.js"
ServerEvents.recipes(event => {
    event.custom({
        type: "create:emptying",
        ingredients: [
            {
                item: "minecraft:wheat"
            }
        ],
        results: [
            {
                item: "minecraft:diamond"
            },
            {
                fluid: "minecraft:water",
                amount: 1000
            }
        ]
    })
})
```

## Mechanical Mixing (`fluid + fluid to fluid mixing`)

**forge 1.20.1**

![super heated](/img/c/fluid_to_fluid.png)



*This recipe will show you how to mix two different fluids into a different fluid as your `output`*



```jsx title="fluid_to_fluid_mixing.js"
ServerEvents.recipes(event => {
    event.custom({
        "type": "create:mixing",
        "ingredients": [
            {
                "fluid": "minecraft:water",
                "amount": 1000 //this is the amount of fluid in buckets, 1000 means 1000mb and 1000mb means 1 bucket. 500mb means half a bucket
            },
            {
                "fluid": "create:chocolate",
                "amount": 1000
            }
        ],
        "results": [
            {
                "fluid": "create:honey",
                "amount": 1000
            }
        ],
        "processingTime": 100
    })
})
```
