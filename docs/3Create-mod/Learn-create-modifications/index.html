Learn Create Modifications forge 1.20.X

Learn default create mod MODIFICATIONS​

There is a few things in the default Create mod you can modify with kubjs and its all very starter friendly. Lets look at some examples of things EVERYONE changes

Machines you can modify​

  • mechanical crafters
  • Millstone
  • Deployers Including
    • sequenced assembly
    • deploying
    • stripping
    • automative crafting

  • Mechanical Saw

    • wood cutting
    • crafting
    • stone cutting
    • glass cutting
    • crafting

  • Item Drain

    • Draining items into fluids and items
  • Mechanical Mixer
    • automated brewing
    • automatic shapless crafting
    • Muxing (crafting)
    • Including heated and non heated recipes
  • Mechanical Press
    • Automated Packing (ITEMS - shaped crafting)
    • Compacting (auto crafting W Items/Fluids)
    • Pressing (things like) -iron sheets
      • dirt into path blocks
      • etc.
  • Encased Fans Including
    • haunting (soul campfire and soul fire)
    • smoking (camp fire and fire)
    • washing
    • blasting
  • and sand paper

Examples of each​

This may look like a LOT of reading, but i promis this will be worth it. We will show you an example of everything here. There should also be different versions of each example of the code to work on multiple different versions sense loads of things change between FORGE, NEOFORGE, and fabric. So be sure to reach wha version this code is for before trying it.

MECHANICAL CRAFTERS​

forge 1.20.1

In this example we show you how to make a very unfair recipe, but fun none the less. You need to fill up a 4X4 set of mechanical crafters to make 1 single oak plank. Very annoying but this is a great recipe

mechanical_crafters.js
ServerEvents.recipes(event => {
    event.custom({
        type: "create:mechanical_crafting",
        pattern: [
            "AAAA",
            "AAAA",
            "AAAA",
            "AAAA"
        ],
        key: {
            A: {
                item: "minecraft:oak_log"
            }
        },
        result: {
            item: "minecraft:oak_planks",
            count: 1
        }
    })
})

Millstone​

forge 1.20.1

this recipe shows you how to have a input to the millestone turn into something as the output. this one specifically shows you putting 1 log into the millestone and getting a plank out as the output

Millstone.js

  "type": "create:milling",

  ],
  "results": [
    {
      "item": "minecraft:oak_planks",
      "chance": 1
    }
  ],
  "processingTime": 1
    })

Draining​

forge 1.20.1

This recipe will show you how to take a item IN to the drain and output two things, You absolutly have to have a fluid output. if you want 2 items the second one has to be a item, not another fluid. The drain can only hold up to one fluid at a time. Just make sure you 100% of the time have some sort of fluid coming out as a result or you will crash your game and the recipe wont work

drain.js
ServerEvents.recipes(event => {
    event.custom({
        type: "create:emptying",
        ingredients: [
            {
                item: "minecraft:wheat"
            }
        ],
        results: [
            {
                item: "minecraft:diamond"
            },
            {
                fluid: "minecraft:water",
                amount: 1000
            }
        ]
    })
})

## Mechanical Mixing (`Non heated`)

**forge 1.20.1**

![normal mixing](/img/c/mechanical_mixing.png)
![normal mixing](/img/c/mixingJEI.png)


*This recipe will show you how to take a item INTO a basin and mix it with the mechanical mixer `non heated` for a specific output,
this one shows you how to put in `oak_logs` and get `oak_planks` as a result*



```jsx title="mixing.js"
ServerEvents.recipes(event => {
    event.custom({
        "type": "create:mixing",
        "ingredients": [
            {
                "item": "minecraft:oak_log"
            },
            {
                "item": "minecraft:oak_log"
            }
        ],
        "results": [
            {
                "item": "minecraft:oak_planks",
                "count": 1
            }
        ],
        "processingTime": 100
    })
})

Mechanical Mixing (fluid + fluid to fluid mixing)​

forge 1.20.1

This recipe will show you how to mix two different fluids into a different fluid as your output

fluid_to_fluid_mixing.js
ServerEvents.recipes(event => {
    event.custom({
        "type": "create:mixing",
        "ingredients": [
            {
                "fluid": "minecraft:water",
                "amount": 1000 //this is the amount of fluid in buckets, 1000 means 1000mb and 1000mb means 1 bucket. 500mb means half a bucket
            },
            {
                "fluid": "create:chocolate",
                "amount": 1000
            }
        ],
        "results": [
            {
                "fluid": "create:honey",
                "amount": 1000
            }
        ],
        "processingTime": 100
    })
})

Encased fan​

*now its time to learn the one things that people LOVE and also get confused on. Encased Fans can be amazing to use for many different reasons BUT the name spacing for different types of things can confuse people. So here are a few recipe types you can do with these

  • haunting
    • create:haunting
  • washing
    • create:splashing
  • Smoking (Fan + Fire)
    • create:smoking
  • Blasting (Fan + Lava)
    • create:blasting

Haunting​

forge 1.20.1

This recipe will show you how to do haunting and this is by putting a soul fire or soul camp fire infront of the encased Fans and getting a output of some sort. this example shows Oak_logs going in and oak_planks coming out

haunting.js
ServerEvents.recipes(event => {
    event.custom({
        "type": "create:haunting",
        "ingredients": [
            {
                "item": "minecraft:oak_log"
            }
        ],
        "results": [
            {
                "item": "minecraft:oak_planks",
                "count": 1
            }
        ]
    })
})

Crushin Wheels​

forge 1.20.1

This recipe will show you how to do crushing and this when you have two crushing wheels spinning inwards very quick and put two items in and get some items out. This is typically used for automatic ore creation or duplication

            {
                "item": "minecraft:oak_planks",
                "count": 4
            },
            {
                "item": "minecraft:oak_planks",
