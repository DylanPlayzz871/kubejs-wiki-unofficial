Learn default create mod MODIFICATIONS

There is a few things in the default Create mod you can modify with kubjs and its all very starter friendly. Lets look at some examples of things EVERYONE changes

Machines you can modify​

  • mechanical crafters
  • Millstone
  • Deployers Including
    • sequenced assembly
    • deploying
    • stripping
    • automative crafting

  • Mechanical Saw

    • wood cutting
    • crafting
    • stone cutting
    • glass cutting
    • crafting

  • Item Drain

    • Draining items into fluids and items
  • Mechanical Mixer
    • automated brewing
    • automatic shapless crafting
    • Muxing (crafting)
    • Including heated and non heated recipes
  • Mechanical Press
    • Automated Packing (ITEMS - shaped crafting)
    • Compacting (auto crafting W Items/Fluids)
    • Pressing (things like) -iron sheets
      • dirt into path blocks
      • etc.
  • Encased Fans Including
    • haunting (soul campfire and soul fire)
    • smoking (camp fire and fire)
    • washing
    • blasting
  • and sand paper

Examples of each​

This may look like a LOT of reading, but i promis this will be worth it. We will show you an example of everything here. There should also be different versions of each example of the code to work on multiple different versions sense loads of things change between FORGE, NEOFORGE, and fabric. So be sure to reach wha version this code is for before trying it.

MECHANICAL CRAFTERS​

forge 1.20.1

In this example we show you how to make a very unfair recipe, but fun none the less. You need to fill up a 4X4 set of mechanical crafters to make 1 single oak plank. Very annoying but this is a great recipe

mechanical_crafters.js
ServerEvents.recipes(event => {
    event.custom({
        type: "create:mechanical_crafting",
        pattern: [
            "AAAA",
            "AAAA",
            "AAAA",
            "AAAA"
        ],
        key: {
            A: {
                item: "minecraft:oak_log"
            }
        },
        result: {
            item: "minecraft:oak_planks",
            count: 1
        }
    })
})

Millstone​

forge 1.20.1

this recipe shows you how to have a input to the millestone turn into something as the output. this one specifically shows you putting 1 log into the millestone and getting a plank out as the output

Millstone.js
ServerEvents.recipes(event => {
    event.custom({
  "type": "create:milling",
  "ingredients": [
    {
      "item": "minecraft:oak_log"
    }
  ],
  "results": [
    {
      "item": "minecraft:oak_planks",
      "chance": 1
    }
  ],
  "processingTime": 1
    })
})

Deploying​

forge 1.20.1

this recipe shows you how to have 1 item as the input (deployer holding) and another item on the belt and how to get a output from all of that. This shows you how to take a Log being held, deploying it onto another Log and getting a plank as the output

deploying.js
ServerEvents.recipes(event => {
    event.custom({
        "type": "create:deploying",
        "ingredients": [
            {
                "item": "minecraft:oak_log"
            },
            {
                "item": "minecraft:oak_log"
            }
        ],
        "results": [
            {
                "item": "minecraft:oak_planks",
                "count": 1
            }
        ]
    })
})

Draining​

forge 1.20.1

This recipe will show you how to take a item IN to the drain and output two things, You absolutly have to have a fluid output. if you want 2 items the second one has to be a item, not another fluid. The drain can only hold up to one fluid at a time. Just make sure you 100% of the time have some sort of fluid coming out as a result or you will crash your game and the recipe wont work

drain.js
ServerEvents.recipes(event => {
    event.custom({
        type: "create:emptying",
        ingredients: [
            {
                item: "minecraft:wheat"
            }
        ],
        results: [
            {
                item: "minecraft:diamond"
            },
            {
                fluid: "minecraft:water",
                amount: 1000
            }
        ]
    })
})
